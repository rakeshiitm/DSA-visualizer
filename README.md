# DSA Visualizer

A simple tool to visualize common Data Structures and Algorithms.

## Features
- Pathfinding: Visualize BFS and DFS on an interactive grid. Draw walls by clicking and dragging.
- BST: Visualize Binary Search Tree insertion on a canvas with automatic spacing.
- Sorting: Watch Bubble Sort and Insertion Sort in action with adjustable speed.

## How to Run
1. Open `index.html` in your browser.
2. Choose a visualizer from the menu.

## Tech Used
- HTML, CSS, and Vanilla JavaScript.
- Promises and Async/Await for animations.
